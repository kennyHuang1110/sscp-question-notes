# SSCP Exam Questions

This repository keeps SSCP practice questions in a Git-friendly format.

## Contents

- `sscp_all_topics_questions_grouped.docx` - original Word document.
- `questions/sscp_questions.md` - all questions converted to Markdown.
- `questions/topics/` - one Markdown file per topic.
- `notes/` - study notes and concept summaries by domain.
